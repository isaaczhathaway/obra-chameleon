# Changelog

## v0.2 - 2026-07-14

- Added the modular two-color Obra Chameleon shader, diagnostic shaders, and five presets.
- Made the author's warm 1280x720 configuration the default preset.

- Added the Windows CMD launcher and persistent PowerShell success/error prompt.
- Pinned official ReShade 6.7.3 download and runtime checksums.
- Added package manifests, payload checksums, dry-run support, and clearer diagnostics.
